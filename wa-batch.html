Ready to start. Enter phone numbers and click "Start Batch Process".

Usage Tips

  • Phone numbers can be in any format: +91-9876543210, 919876543210, or 9876543210
  • Use country code override for local numbers (e.g., 91 for India)
  • The message will be pre-filled in WhatsApp - you can edit it before sending
  • Each click opens a new WhatsApp chat tab in your browser
  • Make sure WhatsApp Web is logged in before starting
  • You can add images or files in WhatsApp after the message loads
