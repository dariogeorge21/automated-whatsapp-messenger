Tips for best results

  • Phone numbers can be in any format: +91-9876543210, 919876543210, or 9876543210.
  • Use country code override for local numbers (e.g., 91 for India).
  • The message will be pre-filled in WhatsApp - you can edit it before sending.
  • Make sure WhatsApp Web is logged in before starting the automation.
  • PyAutoGUI backend must be running for automation steps (Paste, Send, Close) to work.
